📝 Comment utiliser cette fiche

  1. Pour chacune des 20 user stories ci-dessous, lisez le critère d'acceptation (ce qui doit être vrai une fois le test réussi).
  2. Rédigez vous-même, dans vos propres mots, le prompt que vous donneriez à Claude Code pour obtenir ce test — champ "✍️ Votre prompt".
  3. Donnez ce prompt à Claude Code, laissez-le écrire et exécuter le test.
  4. Copiez-collez le code du test qu'il a généré dans le champ "💻 Script Playwright obtenu".
  5. Cochez le résultat (✅ Passé / ❌ Échoué) et notez une observation dans "📋 Rapport".
  6. Une fois les 20 cas traités, cliquez sur "Exporter mon rendu" en bas de page pour télécharger votre fiche complétée.
💡 Vous êtes libre de la formulation : il n'y a pas un seul bon prompt. L'objectif est d'apprendre à traduire un besoin (la user story) en instruction claire pour un agent de test — pas de deviner une phrase "attendue".
📘 Un corrigé de référence existe (EXERCICE-CLAUDE-CODE-SAUCEDEMO-20-ETAPES.html) avec un prompt et un code déjà vérifiés pour chaque cas. Consultez-le après avoir essayé par vous-même, pour comparer votre approche — pas avant.
