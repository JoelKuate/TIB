📌 Comment ce rapport est-il produit ?

Avec Claude Computer Use, le testeur ne code pas de script Playwright — il donne des prompts en langage naturel à Claude qui pilote le navigateur. Une fois l'exécution terminée, Claude fournit une réponse textuelle décrivant ce qu'il a vu et vérifié. Le testeur consigne ces résultats dans ce document.
